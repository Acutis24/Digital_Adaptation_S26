# Acutis24.github.io

A simple GitHub Pages website that loads your essay from essay.txt.

## How to use

1. Open essay.txt in this repository.
2. Replace the placeholder text with your essay.
3. Save and commit the changes.
4. Visit the site in your browser or refresh the page.

## Files

- index.html — page layout and website structure.
- styles.css — page styling.
- scripts.js — loads and renders essay content.
- essay.txt — your essay file.
